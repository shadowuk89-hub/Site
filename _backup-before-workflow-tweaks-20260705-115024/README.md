# Video Editor Premium Portfolio

Статический сайт-визитка для видеомонтажера в темном cinematic стиле.

## Контент

- Работы и ссылки редактируются в `main.js` в массиве `portfolioItems`.
- Telegram редактируется в `main.js` в объекте `contact`.
- Hero-изображение лежит в `assets/cinematic-hero.png`.

## Проверка

```bash
npm run build
```

Сайт можно открыть напрямую через `index.html`.
