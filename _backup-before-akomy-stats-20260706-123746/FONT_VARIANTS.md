# Font variants

Current active variant is set in `styles.css` inside `:root`.

To return to the original font, set:

```css
--font-body: var(--font-original);
--font-display: var(--font-original);
--font-ui: var(--font-original);
```

Available local variants:

```css
--font-body: var(--font-variant-clean);
--font-display: var(--font-variant-clean);
--font-ui: var(--font-variant-clean);
```

```css
--font-body: var(--font-variant-editorial);
--font-display: var(--font-variant-editorial);
--font-ui: var(--font-variant-editorial);
```

```css
--font-body: var(--font-original);
--font-display: var(--font-variant-sharp);
--font-ui: var(--font-original);
```
